Works With Many Devices

The whole API is HTML & javascript based so any device
that has a suitable browser will work.  This includes the
latest smartphones or games consoles which contain built
in browsers, stand alone kiosks or computers running any
modern browser.

You can even run the Differences API from a usb memory
stick avoiding messy installations.  Just drop your
level into the browser window or address bar to start
- perfect for on the go promotions.

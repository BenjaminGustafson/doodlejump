FAQ's

Q) Can I use the Differences API with other tools such as AppMobi?
A) Yes, just add your game within an iframe and reference that iframe in the AppMobi project body tag.

Q) Can I use jQuery with the Differences API - e.g for cool effects?
A) Yes, import jQuery and any other modules into your game file as normal.
